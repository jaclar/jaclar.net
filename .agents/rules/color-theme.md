# Color Palette & Theme Preferences

- **Primary Highlight Color**: Green / Emerald (`#10b981`, `var(--accent-primary)`, `var(--accent-emerald)`). Used for main interactive highlights, active badges, route lines, primary glows, and key UI accents.
- **Secondary Accent Color**: Light Blue / Cyan (`#38bdf8`, `var(--accent-secondary)`, `var(--accent-cyan)`). Used for secondary highlights, links, code emphasis, and subtle support accents.
