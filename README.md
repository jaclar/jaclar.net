# jaclar.net — Personal Website

The personal website of **Lars Jacob** — software engineer, adventurer, and sailor aboard **S/V Samurai**.

## 🚀 Features

- **Obsidian Dark Space Theme**: Modern UI built with glassmorphism panels, glowing accents, and high-performance layout.
- **Monospace Terminal Engine**: Typewriter effect rendering `introduction.md` with monospace styling, flashing cursor, and Fast Forward controls.
- **Voyage Map Preview**: Non-interactive Leaflet map displaying real GPX voyage tracks with gradient fade masks.
- **Navigation Drawer & Socials**: Compact slide-out menu with direct links to apps (Travel Log) and social profiles (NoForeignLand, Instagram, GitHub, LinkedIn, Email).
- **Responsive & Mobile-Optimized**: Tailored mobile viewport layout with initial map glimpse peeking and SVG scroll indicators.

## 🛠️ Technology Stack

- **Core**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Package Manager**: `pnpm`
- **Map Library**: [Leaflet.js](https://leafletjs.com/) with Leaflet-GPX
- **Typography**: `JetBrains Mono` & `Outfit` (via Google Fonts)

## 💻 Local Development

```bash
# Install dependencies
pnpm install

# Start local development server
pnpm run dev

# Build production bundle
pnpm run build

# Preview production build
pnpm run preview
```
